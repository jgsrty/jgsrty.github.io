# 中国现代文学史-总

# 重点人物

# 鲁迅

批判与同情

# 《狂人日记》的思想主题与内容

  • 描写了**“迫害狂”患者的精神状态和心理活动**
  • 抨击了封建家族制度和封建礼教“吃人“的本质
  • 文学上彻底的反封建
  • 表现出强烈的反抗和变革精神

# 《狂人日记》的艺术成就和特点

  • 艺术表现：冲破了传统的手法
  • 现实主义与象征主义相结合
    • 现实：塑造狂人的形象
    • 象征：肉体吃人上升到封建礼教吃人

# 《阿Q正传》中阿Q的思想性格特征

  • 阿Q的身份特征和生活处境
    • 阿Q是一个深受封建观念侵蚀和毒害的贫苦农民，带有小生产者落后、不觉悟的特点
    • 阿Q的不觉悟更突出的表现在他对“革命”的幼稚、错误的认识上
  • 精神胜利法
    • 阿Q经常遭受挫折和屈辱，但凭着精神胜利法总能得意而满足
  • 意义
    • 暗示了辛亥革命失败的历史教训：没有真正的唤醒民众

# 《呐喊》《彷徨》的思想内容和艺术成就

  • 具有鲜明的反封建思想内涵
  • 通过对农民、妇女、知识分子形象与命运的表现，批判了国民性
  • 开创了多种创作方法的源头，一方面大胆借鉴了西洋小说的表现手法，另一方面又融合了中国传统小说的长处，从而开创和奠定了中国新文学的现实主义传统

# 《故事新编》的艺术特点

  • 依据古籍，容纳现代，在历史故事的基础上进行艺术加工和改造
  • 将古人写活，达到古今交融的艺术效果
  • 运用油滑的手段，对人物赋予现代化的生活细节，为借古讽今服务
  • 举例：《铸剑》《奔月》

# 《野草》在艺术上的探索及成就

  • 构思上：诗与散文相结合，内容含蓄、凝练，具备诗的构思和意境
  • 艺术风格上：哲理性、象征性和形象性相结合
  • 语言上：言尽而意永，把散文诗的抒情和诗的意蕴发挥到了极致
  • 手法上：大量运用象征、隐喻等手法
  • 举例：《秋叶》《死火》《影的告别》

# 《朝花夕拾》的艺术风格

  • 《朝花夕拾》是鲁迅带有回忆性质的散文集
  • 叙事、议论、抒情的有机结合
  • 清新恬淡与讽刺幽默相结合
  • 举例：《从百草园到三味书屋》《藤野先生》《二十四孝图》

# 鲁迅的杂文

  • 前期：热风吹坟，华盖而已
    • 《热风》《坟》《华盖集》《华盖集续集》《而已集》
  • 杂文的艺术特色
    • 创造了许多类型形象
    • 多姿多彩的语言
    • 篇章短小精悍，笔墨凝练犀利
    • 举例：《记念刘和珍君》

# 郭沫若

浪漫与激情
五四时期受西方哲学泛神论的影响
诗集：《瓶》《女神》《前茅》《恢复》《星空》
1921年《女神》的出版，为诗坛开了浪漫新风，也为中国新诗开辟了一个崭新的时代

# 《女神》的思想内容

  • 体现了**“五四”狂飙突进的时代精神**
  • 表达了诗人呼唤新世界诞生的民主理想
  • 对个性解放的强烈要求，对自然的礼赞
  • 显示了彻底破坏和大胆创新的精神
  • 举例：《凤凰涅槃》《天狗》

# 《女神》的艺术成就

  • 《女神》是一部充满浪漫主义色彩的作品
  • 采用了内在情感强化以及喷发式宣泄的表达方式，达到极致的夸张
  • 语言上华赡丰美
  • 主张绝对的自主和自由
  • 创作形式自由多变，大量采用比喻、象征等手法

# 茅盾

心理与现实
农村三部曲《春蚕》《秋收》《残冬》
中篇小说《林家铺子》
《腐蚀》—— 赵惠明，日记体形式，国民党特务统治
《子夜》的问世，标志着中国现代长篇小说发展的成熟

# 茅盾《蚀》三部曲的主要内容

  • 《幻灭》
    • 小资产阶级静女士，对个性解放从追求到幻灭
  • 《动摇》
    • 知识分子方罗兰，男，生存环境的再现
  • 《追求》
    • 大革命失败后，知识分子章秋柳人生追求的悲剧

# 《野蔷薇》的主要内容和艺术价值

  • 小说以恋爱为题材，通过对青年知识分子生活苦闷，寄希望于爱情，最终在迷茫中盘旋的心理状况的描写，表现追求与幻灭的主题
  • 艺术价值
    • 借鉴了现代西方文学的表现手法
    • 细腻真切的心理描写
    • 运用环境对人物心理进行烘托渲染

# 《子夜》

  • 思想意义
    • 《子夜》展示了一幅30年代中国全景式生活图画
    • 表现了当时社会错综复杂的阶级关系和社会矛盾
    • 全面展示了中国民族资产阶级衰败的过程
  • 人物形象
    • 吴荪甫：民族资产阶级，失败的英雄。性格果断、自信；动摇、悲观
    • 赵伯韬：买办资本家。象征剥削阶级，心狠手辣
    • 屠维岳：资本家的走狗
    • 冯云卿：乡下地主。追求利益出卖女儿

# 茅盾的创作成就及对中国现代小说发展的贡献

  • 茅盾小说最大的特点是富于时代色彩，侧重对社会做全景式的描绘
  • 结构恢弘扩大
  • 遵循现实主义的表现手法，客观描写和再现现实生活
  • 为中国现代小说的成熟和发展做出了突出贡献
  • 在茅盾的影响下，现文史上出现了一批追随其创作风格的作家，被称为**“社会剖析派”**

# 巴金

真情与反射
早期受无政府主义思想影响
第一部自传色彩作品**《灭亡》**，杜大心：恨人类
《激流三部曲》：《家》《春》《秋》
《爱情三部曲》：《雾》《雨》《电》
《寒夜》《憩园》

# 《激流三部曲》的主要思想内容

  • 控诉封建家庭制度
  • 喊出了青年一代的呼声
  • 以批判旧家庭制度为窗口，进而对整个旧社会旧制度进行了批判

# 《激流三部曲》的艺术特点

  • 作品具有很高的典型化程度
  • 运用抒情的方式，着重发掘人物的内心世界
  • 结构上以事件为主线索，以场面串联故事
  • 寄托了作家强烈的道德批判

# 《激流三部曲》中人物形象的塑造和意义

  • 高觉慧
    • 性格富有叛逆性
    • 行为既大胆又幼稚
    • 这个形象表明，只有革命才是唯一的出路
  • 高觉新
    • 觉新是贯穿全书始终的中心人物
    • 对封建家庭制度俯首听命，内心无法压抑对自由的渴望
    • 矛盾而病态的封建家庭制度的牺牲品
    • 典型的多余人形象
  • 高老太爷
    • 封建家长制的代表
    • 突出了他性格的专横、虚伪和孤独感
    • 性格的复杂性，避免了人物的脸谱化

# 《寒夜》

  • 《寒夜》是最能代表巴金40年代创作风格的长篇力作
  • 通过一个小公务员在生活的重压下经历家庭破裂的悲剧，来揭示旧中国知识分子的命运，暴露了抗战后期国统区的黑暗现实
  • 汪文宣，是一个被侮辱、被损害的形象
  • 曾树生，是一种反道德、重自我的新型女性形象
  • 艺术特色
    • 细腻入微地挖掘人物的内心世界
    • 作品风格由热到冷
    • 笔调冷峻、气氛肃杀，给人一种冬夜的寒气

# 《憩园》

  • 主要内容
    • 以憩园公馆为背景，表现了杨、姚两家的悲剧命运
    • 旧主人杨老三败尽祖上家产，又不屑自食其力，最终酿成人生悲剧
    • 新主人姚家父子靠遗产度日，最终走向破落
    • 揭示了腐朽的封建制度对人性的扭曲和毒害
  • 艺术风格
    • 通篇蒙上凄美、抒情的调子
    • 构思别具一格，两个故事互相映衬，构成复调关系
    • 设置了故事的总叙述着“我”

# 老舍

京味与幽默
英国任教时期所写:《老张的哲学》《赵子曰》《二马》
抗战爆发前：《猫城记》科幻小说，《离婚》标志着老舍创作思想艺术的新高度 新中国成立前：《鼓书艺人》，《月牙儿》：母女两代被迫沦为暗娼的悲剧

# 《月牙儿》《微神》《断魂枪》等短篇小说的特点

  • 共同点：散文诗的笔法、象征主义
  • 《断魂枪》：白描的手法，表现人物性格和内心世界

# 《骆驼祥子》

  • 最初连载与**《宇宙风》杂志**
  • 祥子、虎妞、小福子、刘四爷
  • 骆驼祥子是老舍小说创作的代表作和里程碑
  • 主要内容：通过描写旧北京人力车夫祥子的悲剧，表达了对挣扎在社会底层的劳动人民悲惨命运的深切关怀和同情

# 虎妞的形象

  • 她有着剥削阶级好逸恶劳、善玩心计、粗俗刁泼的性格特点
  • 对祥子的爱是以不平等的态度体现，更多是畸形的性纠缠和索取，对祥子心灵和肉体的双重摧残
  • 破坏了祥子的个人奋斗目标，是祥子走向堕落的诱因之一

# 造成祥子悲剧的原因

  • 主观原因
    • 思想上的局限性与心理上的弱点
    • 小农意思、目光狭隘
    • 体力上的强者、心灵上的弱者
  • 客观原因
    • 把人变成鬼的旧社会的迫害
    • 买车的愿望和历尽艰辛，买车三起三落的经历
