# 美学-审美形态论

## 审美形态的内涵和特征
#### 划定审美形态的标准
- 首先是**广泛性和普适性**。适用于各种艺术体裁和生活中的审美现象
- 其次是**统摄性**。集杂多于统一
- 最后是**历史性**。有些审美形态积淀于民族审美文化历史中，产生了深远的影响

#### 审美形态的特征 ⭐️
- 生成性：与历史和个人相关
- 贯通性：民族文化
- 二重性：多种审美因素构成的感性凝聚
- 兼容性：民族性与世界性的统一
